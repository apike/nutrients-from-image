# Nutrients from Image

A very simple service that returns the nutrients from an image of a nutrition facts label, if one is present.

## Available Scripts

In the project directory, you can run:

### `pnpm run dev`

To start the app in dev mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `pnpm start`

For production mode

### `pnpm run test`

Run the test cases.
